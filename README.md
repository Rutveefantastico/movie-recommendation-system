# 🎬 Movie Recommendation System + AI Dashboard

🚀 A complete **Movie Recommendation Web App** with an **AI-powered analytics dashboard**.

This project combines **Machine Learning, Web Development, and AI-assisted Data Visualization** to deliver both recommendations and insights from movie data.

---

## 📌 Project Overview

This system recommends movies similar to a selected movie using a **content-based filtering approach**.  

Additionally, an **interactive dashboard** was created using AI-assisted development to analyze movie trends and patterns.

---

## ✨ Features

### 🎬 Recommendation System
- Content-based movie recommendation
- Cosine similarity algorithm
- Searchable movie database
- Netflix-style UI
- Movie posters, ratings & trailers (TMDB API)
- User login system
- Watchlist feature (Add/Remove movies)

---

### 📊 AI-Powered Dashboard
- Built using **AI-assisted development (Claude AI)**
- Visualizes:
  - 📈 Movies released over time
  - 🎭 Genre distribution
  - ⭐ Rating distribution
  - 🔥 Popularity (vote count)
- Interactive and user-friendly design

---

## 🧠 Technologies Used

- **Python**
- **Pandas & NumPy**
- **Scikit-learn (Cosine Similarity)**
- **Streamlit (Web App)**
- **TMDB API (Movie Data)**
- **HTML / CSS / JavaScript**
- **Claude AI (Dashboard Design Assistance)**

---

## ⚙️ How It Works

1. Movie data is preprocessed and combined into tags
2. Text data is vectorized using CountVectorizer
3. Cosine similarity is computed between movies
4. Top similar movies are recommended
5. TMDB API fetches posters, ratings, and trailers
6. Dashboard visualizes dataset insights
